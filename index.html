DanFlix

Desenvolvido em ASP.NET MVC, o DANFLIX é uma aplicação interativa que utiliza a API TMDb para oferecer informações sobre filmes em cartaz, populares e bem avaliados. Com design responsivo, a plataforma permite pesquisas personalizadas e detalhes aprofundados sobre cada filme.A funcionalidade de detalhes inclui o número de curtidas, orçamento, receita, duração e sinopse, proporcionando aos usuários uma experiência enriquecedora na escolha de filmes.

Listagem de Contatos (ASP.NET MVC)

Este projeto, desenvolvido em ASP.NET MVC, proporciona um sistema para listar, adicionar, editar e apagar contatos. Utilizando um padrão de arquitetura MVC, o código inclui um controlador (ContatoController), uma classe de modelo (ContatoModel), e um repositório (ContatoRepositorio) para interagir com o banco de dados. As visualizações permitem navegar pela lista de contatos, adicionar novos, e realizar operações de edição e exclusão.

Tempo do Dia: Uma Jornada Visual pelas Horas do Dia

Este projeto interativo, intitulado "Hora do Dia", exibe uma mensagem dinâmica e altera uma imagem de fundo com base nas principais horas do dia: manhã, tarde e noite. Ao carregar a página, o script JavaScript calcula a hora atual, ajusta a mensagem para refletir o horário e troca a imagem de fundo correspondente. A interface simples oferece uma experiência visual agradável, mudando conforme as diferentes fases do dia.

Calculadora Web: Realize Operações Matemáticas Rapidamente

Este projeto é uma calculadora web interativa que permite realizar operações matemáticas básicas. A interface inclui botões para números, operadores e funcionalidades como limpar, apagar e calcular. O JavaScript associado aos botões executa as operações correspondentes, atualizando dinamicamente o resultado na tela. Essa calculadora proporciona uma maneira simples e eficaz de realizar cálculos diretamente no navegador.

Login Moderno: Acesse sua Conta com Facilidade

Este projeto consiste em uma tela de login com um design moderno e responsivo. Oferece campos para inserção de e-mail e senha, com ícones indicativos, e um botão para submeter o formulário de login. Além disso, inclui um link para recuperação de senha. A estrutura HTML utiliza tags semânticas, enquanto o CSS está organizado em arquivos externos para melhor organização. A interface é adaptável a diferentes tamanhos de dispositivos, garantindo uma experiência consistente aos usuários.

Cordel Moderno de Milton Duarte: Poesia e Tecnologia

Este projeto é uma página web que exibe um cordel moderno do poeta Milton Duarte, combinando imagens com um efeito Parallax simples para um visual cativante. O cordel aborda a influência da tecnologia na comunicação de forma poética, oferecendo uma experiência envolvente ao alternar entre seções normais e imagens.

A História do Mascote do Android

Este projeto é uma página web que explora a história do mascote do Android, Bugdroid. Desde a criação inicial por Dan Morrill até a versão final por Irina Blok, o artigo destaca curiosidades, inspirações de design e a mudança na nomenclatura das versões do Android. Descubra a evolução do Bugdroid e curiosidades sobre o sistema operacional Android.
